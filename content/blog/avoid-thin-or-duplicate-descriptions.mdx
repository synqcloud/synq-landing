Card stores list huge catalogs, which makes thin and duplicate descriptions a real SEO risk. If every page repeats the same generic blurb, search engines have little reason to rank them. With a Synq [description template](/help/product-templates), you can give every card unique, useful content automatically.

## Why does this matter for a card store?

You might list thousands of cards. If they all share identical text, search engines treat the pages as thin or duplicate content, which holds back rankings. Each page needs something specific to that card.

## How do I make each description unique?

Use a [description template](/help/product-templates) with macros, including the card text macro `{description}`, which inserts the card's own rules text or product description from the catalog. Combined with `{name}`, `{set_name}` and `{rarity}`, every product gets content that is specific to that card, generated from a single template.

## What should a description include?

- The **card text** for singles, which is naturally unique per card.
- Key details like **set**, **rarity** and **printing**.
- What the buyer gets, especially for sealed product.

## Keep the formatting clean

Structured bullet points help shoppers scan, and Synq cleans up the formatting so lists show correctly on your live theme. If a description ever looks off on the storefront, see [storefront formatting and theme issues](/help/storefront-formatting-and-theme-issues).

Pair unique descriptions with [titles that rank](/blog/write-product-titles-that-rank) for the strongest result.
